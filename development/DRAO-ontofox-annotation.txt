#### ONTOFOX IMPORT CONFIGURATION ####
##
## Ontofox allows more than one ontology to be queried at once.
## This file contains the settings for importing subsections of
## the following external ontologies:
##
## BFO, CHEBI, CHEMINF, CHMO, CL, CLO, CMO, DOID, DRON, EDAM,
## EFO, ENVO, ERO, FBBI, FBCV, FMA, GO, HP, IAO, IDO, IDOMAL,
## MAMO, MFOEM, MI, MOD, MP, MS, NCBITaxon, NCIT, OAE, OBCS, OBI,
## OGI, OGMS, OMIT, OMP, PATO, PECO, PO, PR, PW, SBO, SIO, SO, STATO
## SWO, UBERON, UO, VariO, VO
##
## Terms from ontologies that are not accessible from Ontofox are
## imported manually into DRAO-manual.owl
##
##
## General Note:
## Please note that the use of includeComputedIntermediates means
## that a cut-down hierarchy will be imported, where intermediate
## parent classes with only one child are not included. However,
## this may cause issues in future as we are importing a large
## number of external ontologies, each of which would create a
## different computed hierarchy for a single ontology. As an example,
## EFO and GO imports both choose different parts of the GO hierarchy.
## Therefore one import might assert that GO class XXX is a direct child of
## GO class ZZZ, while the other import asserts that XXX is a direct
## child of YYY. This will result in two asserted parents, and will
## need to be managed.
##
##

[URI of the OWL(RDF/XML) output file]

#### NOTES ON BFO IMPORT ####
##
## BFO is indirectly imported by other external ontologies used in
## the AO as well as directly through this configuration file.
## This means that duplicate annotation properties are also introduced.
## We need to solve this issue so the resulting AO looks cleaner, though
## this is not an issue which breaks things.
##
## The includeAllIntermediates setting has been enabled for this
## upper-level ontology so that the correct hierarchy can be used for
## all lower-level ontology imports into the FAIRsharing AO.
## Upper-level ontologies tend to have fewer classes, while
## the includeComputedIntermediates setting is used for other
## imports to reduce bloating of the AO.
##
